## 🎯 Why this framework?

Modern QA Automation is no longer just about writing UI tests.

This project demonstrates how enterprise-grade automation frameworks should be designed using scalable architecture, reusable design patterns, CI/CD, API testing, and AI-assisted engineering practices.

It is being built as a production-ready reference framework that reflects the engineering standards used by modern software teams.

- ✅ Playwright with TypeScript
